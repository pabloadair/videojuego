# Platformer Quest - Construct 3 Project

Videojuego de plataformas desarrollado como proyecto colaborativo para la materia de aplicaciones web.

## Descripción
Este repositorio contiene la lógica de niveles y diseño de mecánicas para un videojuego de estilo plataformas. El proyecto fue desarrollado utilizando el motor Construct 3, enfocándose en el diseño de niveles y experiencia de usuario (UX).

## Logros

Diseño de mecánicas de movimiento y colisiones.

Implementación de activos visuales y lógica de progresión.
